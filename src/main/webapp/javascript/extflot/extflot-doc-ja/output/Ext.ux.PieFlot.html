Class Ext.ux.PieFlot

Package:Ext.ux
Defined In:PieFlot.js
Class:PieFlot
Extends:Flot
円グラフ専用の Flot です。
複数の系列をひとまとめにして、データごとに値をまとめて与えることができます。 
data: [{
  label: ,
  data: [[, ], ...]
},{
...
}]
data の X 値 (data[i][j][0]) には、その系列の ID を入れますが、 これは内部的には無視され、 Y 値 (data[i][j][1]) の和が その系列の値になります。 対象の系列が円グラフで占める割合は、この各系列の値の比になります。 
series: [
  { label: 'Niveau A', data: [[0,2]]},  # 40%
  { label: 'Niveau B', data: [[1,2]]},  # 40%
  { label: 'Niveau C', data: [[2,1]]},  # 20%
  { label: 'Niveau D', data: [[3,0]]}   #  0%
],
円グラフは操作の必要がないので、デフォルトのコンテキストメニューは プロパティのみになっています。
